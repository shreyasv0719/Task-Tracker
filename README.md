# 📝 Task Tracker Web App

A simple and clean Task Tracker built using **HTML, CSS, and JavaScript**.  
This project helps users add, manage, and organize daily tasks in an easy way.

---

## 🚀 Live Demo
👉 https://shreyasv0719.github.io/Task-Tracker/

---

## 📌 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- 📋 Simple and clean UI
- ⚡ shows graph and pie chart as progress
- 💾 Runs directly in the browser (no installation needed)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
